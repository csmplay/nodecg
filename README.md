# CSM NodeCG

## Description

This is an edited version of [NodeCG](https://github.com/nodecg/nodecg) that is
used on tournaments hosted by CSM.

## Extra information

[nodecg-cli](https://github.com/nodecg/nodecg-cli) and
[generator-nodecg](https://github.com/nodecg/generator-nodecg) are included as
packages in this repo

If you want to use them you will need
[`npx`](https://docs.npmjs.com/cli/v9/commands/npx?v=true) command

e.g. `npx nodecg install "BUNDLE_REPO/BUNDLE_NAME"` or `npx yo nodecg`
